Типы сайтов

Landing Page

Лэндинг – одностраничный сайт с продающим контентом. Основная черта это лаконичность. Характеристики:

  • картинки с акциями
  • кнопки быстрого оформления заказа
  • продающий текстовый контент
  • яркие баннеры

Корпоративный сайт

Содержит сведения о компании, принципы работы, описание услуг, контактные данные. Преймущества:

  • формирование имиджа организации
  • поиск заказчиков и инвесторов
  • поиск новых сотрудников
  • оперативная связь с клиентами

Интернет магазин

Сайт, торгующий товарами посредством сети интернет. Cодержит такие компоненты:

  • удобный каталог
  • бесперебойное функционирование
  • простой поиск
  • оригинальный дизайн
  • продающие описания
  • быстрое переключение
